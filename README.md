## Projects

### Exercise 01 - Educational Platform API

Basic JWT authentication system with role-based access control.

**Features:**
- User registration and login
- JWT token generation and validation
- Role-based access control (STUDENT, PROFESSOR, ADMINISTRATOR)
- Permission-based authorization
- Secure endpoints with Spring Security

**Tech Stack:**
- Java 17
- Spring Boot 3
- Spring Security 6
- JWT (jjwt)
- MySQL
- Lombok

### Final Project - Blog Security API

Enterprise-level REST API with complete authentication and authorization system.

**Features:**
- JWT authentication with role-based authorization
- OAuth2 integration (Google & GitHub)
- Custom roles: USER, AUTHOR, ADMIN
- Permission-based access control (CREATE, READ, UPDATE, DELETE)
- BCrypt password encryption
- Docker containerization
- Method-level security with @PreAuthorize

**Tech Stack:**
- Java 17
- Spring Boot 3.5.7
- Spring Security
- PostgreSQL 16.11
- JWT (jjwt 0.12.6)
- OAuth2 Client
- Docker & Docker Compose
- Lombok

**[View project →](./blog-security-api)**
